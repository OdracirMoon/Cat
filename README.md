# 🐈‍⬛ El Tarot de los Gatos

Herramienta interactiva de lectura del Tarot de los Gatos — estilo gótico oscuro.

## 📁 Archivos

| Archivo | Descripción |
|---------|-------------|
| `index.html` | Página principal de la app |
| `style.css` | Estilos góticos con animaciones |
| `app.js` | Lógica + datos de las 78 cartas |

## 🚀 Publicar en GitHub Pages

1. Crea un repositorio en GitHub (ejemplo: `tarot-gatos`)
2. Sube los 3 archivos: `index.html`, `style.css`, `app.js`
3. Ve a **Settings → Pages → Source → main branch → / (root)**
4. GitHub te dará el link: `https://tu-usuario.github.io/tarot-gatos/`

## ✨ Características

- **78 cartas** completas (22 Arcanos Mayores + 56 Arcanos Menores)
- Significado detallado y significado invertida de cada carta
- Filtros por palo: Mayores, Bastos, Oros, Espadas, Copas
- Buscador por nombre o palabras clave
- Diseño gótico oscuro con animaciones
- Totalmente responsive (funciona en móvil y PC)
- Sin dependencias externas (solo fuentes de Google Fonts)
